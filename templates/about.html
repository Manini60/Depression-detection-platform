What Is This?

MindScan is a machine learning-powered web application that predicts the risk of depression among college students using a Bayesian Network model.

By analysing key lifestyle indicators — including sleep patterns, academic pressure, financial stress, dietary habits, and study satisfaction — the system computes a probabilistic assessment of depression risk using Variable Elimination inference.

The model was trained on a real-world survey dataset of over 27,000 Indian students, providing clinically-informed predictions grounded in real data.

How It Works
Data Collection
27,902 student survey records across Indian cities
Preprocessing
Discretisation of continuous variables into categorical bins
Model Training
Bayesian Network fitted using Maximum Likelihood Estimation
Inference
Variable Elimination computes posterior P(Depression | evidence)
